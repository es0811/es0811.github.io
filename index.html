Hi! I am an aspiring Software Developer. I am currently pursuing Masters in CS from Georgia Tech University, US. I also hold a Bachelor's Degree in Electronics and Telecommunication Engineering from Goa College of Engineering, India. Currently, I am based in Atlanta, Georgia and I am looking for internship/ entry level opportunities as a Software Developer. I am most interested in roles which involve coding resourceful software, logical thinking and problem solving.

I have more than 5 years of experience working as a Controls Engineer where I was actively involved in software development of PLC and Scada/HMI systems. This is where I developed a passion for coding and decided to pursue a profession as a Software Developer.

Projects

Chips - in - a - row!

This is a internet-browser friendly game which I developed using HTML, CSS and Javascript. This game was also a milestone in one of my Udemy courses, 'Full Stack Bootcamp- By Jose Portilla'. This game is played by two players alternatively. Each player has to drop a chip in a 9x9 matrix. The player who drops four consecutive chips wins the game!

Tech Stack: Javascript, HTML, CSS

Content Management System

This application is a part of the project I had to implement for the Udemy Course 'The Complete PHP MYSQL Course by Jazeb Akram'. This project is currently a work in progress. With this application, various users can register with the system. All the details are stored in a MySQL database. The users can post blogs, search through the blogs, delete posts, get detailed descriptions about each post etc.

Tech Stack: PHP, SQL, XAMPP Server, Javascript, HTML, CSS, Bootstrap

Source

Grocery Service Application

I and me team developed this application for course 6310 Software Architecture & Design in the Georgia Tech CS Masters Program. The implementation included creating APIs for registering stores, users, selling and purshaing items, user authentication, authorization and a postman interface. We had to design the architecture, interface and data structure modeling.

Tech Stack: Java, MySQL, Postman Collection, JSON, IntelliJ
