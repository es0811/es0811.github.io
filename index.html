Distributed File System

This application is a part of the project I had to implement for the Georgia Tech's "Introduction to Operating Systems". This application is very similar to dropbox, where the latest copy of the file is available throughout the multiple clients and the server system. This project introduced us to gRPC, a file tranfer protocol which was developed by google. It also gave us an insight to multi-threading, using of semaphores to achieve isolation and sychronization.
